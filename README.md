#  Face Attendance System

A smart attendance system built using **Python  and OpenCV ** that automatically
marks attendance using real-time face detection and stores it in an Excel file.

##  Features
- Real-time face detection using Haar Cascade
- Automatic attendance marking
- Excel-based attendance storage 
- Date & time logging 
- Sound confirmation 

## Technologies Used
- Python
- OpenCV
- Pandas
- NumPy
- OpenPyXL
